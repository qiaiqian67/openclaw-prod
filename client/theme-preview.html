3 套规范摘要 · A apple = 纯白 + Action Blue #0066cc(单强调色,博物馆级留白,17px body 慢阅读节奏) · B claude = 暖米 #faf9f5 + 珊瑚 #cc785c + 深海军 #181715(衬线 display + 人文 sans,文学感,三表面 trinity) · C minimax = 纯白 + 黑 pill #0a0a0a 强调 + 极简 hairline(80px hero,DM Sans,高密度文档网格)

关键差异:从色相/明度看,apple 是中性冷蓝、claude 是暖色珊瑚、minimax 是高对比黑白。从情绪看:apple = 现代克制、claude = 文学编辑、minimax = 技术密度。三套都跟现有 dark/light 显著区分。挑 2 个告诉我。
